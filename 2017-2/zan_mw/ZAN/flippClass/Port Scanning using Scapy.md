# Port Scanning using Scapy #
## 实现功能 ##
- TCP connect scan
- TCP stealth scan
- TCP XMAS scan
- UDP scan

## 实验环境 ##
- 操作系统：kali linux
- 编程语言：Python 2.7.13
- kali-Gateway扫描kali-1的端口
- kali-Gateway IP:192.168.10.1 (内部网络)
- kali-1 IP:192.168.10.9 (内部网络)
